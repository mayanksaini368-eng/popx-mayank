# 🚀 PopX UI 

A responsive frontend UI project built using React, Vite, and Tailwind CSS. This project is inspired by an existing PopX UI design and enhanced with improvements and customizations.

---

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* React Router

---

## ✨ Features

* Clean and responsive UI
* Login & Signup pages
* Account settings page
* Smooth navigation using React Router
* Improved layout and styling

---

## 🔧 Improvements Made

* UI spacing and alignment improved
* Removed unwanted scrolling issues
* Responsive design optimized
* Code structure cleaned and organized
* Better user experience
